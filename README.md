#Crazy Alert

Snazz up your alerts with CRAAAAZZZYYY ALERT!

##Usage

```javascript
  // Load in with require
  define(function(require){
    var crazyAlert = require('gb-crazy-alert');
    crazyAlert("ayoooooo!");
    /* Output:
      WHOAAAAAA!!!!!!!!!!!!

      ayooooooo!
    */
  });
```

##Installation

```
  npm install gb-crazy-alert
```
